# ZSFlowEditor

#### Description
使用 vue3 、Vite 、 AntV X6 完成的流程设计器。设计器主要的用途为ERP类项目的流程审批。因为平台旧版本使用的SilverLight流程设计器，然后按照项目需求和原有的格式制作的一摸一样的设计器。支持 职能带、节点属性、边条件表达式等。
